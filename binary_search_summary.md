```python
def prefix_function(s):
    n = len(s)
    pi = [0] * n
    for i in range(1, n):
        j = pi[i - 1]
        while j > 0 and s[i] != s[j]:
            j = pi[j - 1]
        if s[i] == s[j]:
            j += 1
        pi[i] = j
    return pi
```

- –°–ª–æ–∂–Ω–æ—Å—Ç—å: **O(n)**

## üßÆ –ê–ª–≥–æ—Ä–∏—Ç–º –õ–µ–≤–µ–Ω—à—Ç–µ–π–Ω–∞ (—Ä–µ–¥–∞–∫—Ü–∏–æ–Ω–Ω–æ–µ —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ)

### üîß –î–∏–Ω–∞–º–∏—á–µ—Å–∫–æ–µ –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏–µ

–û–±–æ–∑–Ω–∞—á–∏–º:

–†–µ–∫—É—Ä—Ä–µ–Ω—Ç–Ω–æ–µ —Å–æ–æ—Ç–Ω–æ—à–µ–Ω–∏–µ:

### ‚ö° –û–ø—Ç–∏–º–∏–∑–∞—Ü–∏—è –ø–æ –ø–∞–º—è—Ç–∏

- –í—Ä–µ–º—è: `O(n * m)`

### üíª –†–µ–∞–ª–∏–∑–∞—Ü–∏—è

```python
def levenshtein(a, b):
    if len(a) < len(b):
        a, b = b, a

    prev = list(range(len(b) + 1))
    curr = [0] * (len(b) + 1)

    for i in range(1, len(a) + 1):
        curr[0] = i
        for j in range(1, len(b) + 1):
            cost = 0 if a[i - 1] == b[j - 1] else 1
            curr[j] = min(
                curr[j - 1] + 1,      # –≤—Å—Ç–∞–≤–∫–∞
                prev[j] + 1,          # —É–¥–∞–ª–µ–Ω–∏–µ
                prev[j - 1] + cost    # –∑–∞–º–µ–Ω–∞
            )
        prev, curr = curr, prev

    return prev[len(b)]
```
